Technical mentor
================

Summary
-------

The technical mentor helps others do projects.  Really, everyone does
this, but it's important enough (and has enough specialized skills)
that it deserves an entry here so that people can get credit for doing
it and reserve time for it.


Main challenges/pitfalls
------------------------

* Mentoring is hard and a different job from doing things yourself.
